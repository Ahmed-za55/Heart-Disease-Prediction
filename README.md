# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview

This project predicts whether a person has heart disease using Machine Learning techniques.

---

## 📊 Dataset

* Source: Kaggle
* 1025 records
* 13 features + target

---

## 🤖 Models Used

* Logistic Regression → Accuracy: ~80%
* Random Forest → Accuracy: ~98% 🔥
